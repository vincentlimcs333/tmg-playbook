Operating Rules
  • Score the development before committing any resources
  • Diagnose the owner before building the offer package
  • Gate must resolve before Phase 2 unlocks
  • Offer package uploaded before any event goes live
  • Every event tracked to hot leads, not just attendance
  • No show unit without a Go from the ROI calculator
  • Every procurement item signed off before distribution
  • Status report runs on real numbers only
  • Campaign closes only after the post mortem is complete

Phase 2 · Step 5
Show Unit ROI Calculator
Establish projected revenue, investment required, and breakeven units. Go/No-Go rule: investment must be below projected units × RM 2,500 cost per sale (or below 4% × avg ticket × units if ticket size exceeds RM 120,000). Timing check: enter the estimated VP end date — the system adds a 2-month defect buffer and calculates how many active selling weeks remain after the show unit is ready. Minimum 8 weeks required.
